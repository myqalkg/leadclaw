# Lead-Claw (领爪) 产品需求文档 (PRD v1.0)

## 1. 产品概述
**Lead-Claw (领爪)** 是一款基于 AI 驱动的微信生态 CPS 爆品分发平台。通过实时监控全网（淘宝、京东、拼多多等）高佣金、高销量的“神价”商品，利用 AI 自动化处理素材，通过微信小程序精准触达用户，实现高效的 CPS 转化。

## 2. 核心价值主张 (Value Propositions)
- **发现价值**：AI 24h 监控“瞬时爆单”和“大额降价”，挑选真正值得买的商品。
- **触达价值**：打破信息差，将全网最赚钱/最省钱的信息第一时间推送到微信。
- **效率价值**：全自动转链与素材生成，降低运营成本。

## 3. 核心功能模块 (Product Features)

### 3.1 智能化全网猎人 (AI Hunter - 后端)
- **实时监控**：对接蜜源等爆品 API，每 5-10 分钟同步一次。
- **神价算法**：依据 [近2h销量增长率]、[佣金比例 (>15%)]、[价格力 (低于历史均价)] 进行综合评分。
- **自动转链**：识别商品 ID 后，自动调用电商联盟 (淘宝/京东/PDD) 接口，生成带 PID 的推广链接。

### 3.2 领爪精选控制台 (Mini-app Frontend - 前端)
- **实时爆品广场**：展示当前全网最火的 Top 50 实时榜单（瀑布流）。
- **分类属性筛选**：支持“美食薅羊毛”、“居家百货”、“数码神价”等维度。
- **详情页**：展示商品原价、券后价、预估省钱金额，展示 AI 生成的“必买理由”。

### 3.3 订阅与精准推送 (Notification)
- **捡漏订阅**：用户可订阅特定关键字或分类（如“手机”、“零食”）。
- **微信触达**：触发神价后，通过微信订阅消息发送卡片。
- **飞书/钉钉同步(可选)**：为高端/B端用户提供实时同步推送。

## 4. 商业与变现模型 (Revenue)
- **CPS 分成**：用户通过点击跳转下单，系统自动结算联盟佣金（预计收益率 5%-20%）。
- **会员制 (VIP)**：提供专属“神价快线”，比普通用户提前 5 分钟收到推送。

## 5. 迭代规划 (Roadmap)
- **Phase 1 (MVP)**：完成 API 对接、数据库入库、后端模拟转链、微信推送闭环。
- **Phase 2**：引入 AI 文本摘要（生成吸引人的文案）、支持更多 CPS 源（美团外卖、电影票）。
- **Phase 3**：私有化部署支持，允许 KOL 接入自己的 PID 进行分发。
